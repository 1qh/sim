# Derive Control in K-map

The headline link: the Control unit is nine Boolean functions of the opcode bits. Each one can be minimized in a K-map, and the minimized expression IS the actual logic wired into the datapath.

## 1. The RegDst truth table

RegDst as a function of the opcode bits across the locked instruction set. R-type sets it 1; loads/stores/branches set it 0 or don’t-care.
